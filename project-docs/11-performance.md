# 性能规范

## 前端

- 首屏 < 1.5s，TTI < 3.5s
- 图片用 WebP/AVIF + 懒加载
- JS 路由级分割，第三方库按需引入
- 列表 > 100 条用虚拟滚动
- 动画优先用 transform/opacity

## 后端

- API P95 < 200ms
- 查询 < 100ms，复杂报表 < 1s
- 数据库查询必须有索引，禁止 N+1
- 深分页用游标分页
- 大文件用流式处理，禁止读入内存
- 外部调用必须设超时
- 长任务必须异步化

## 禁止

- 循环中执行数据库查询
- 一次性读取大文件到内存
- 引入整个库只用 1 个函数
- 无虚拟化渲染万条列表
- 图片不压缩/不懒加载
- API 返回 > 1MB 的 JSON
